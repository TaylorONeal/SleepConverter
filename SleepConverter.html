How to upload to SleepHQ
1. Open SleepHQ and go to the day you want to add data to. 2. Use the O2 Ring / oximeter import and select the CSV you just downloaded. 3. The file is named with the recording start time (YYYYMMDDHHMMSS.csv) and uses the Wellue/Viatom O2 Ring column layout SleepHQ recognises.
